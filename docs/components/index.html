What is this?

This tool is a reference for POLITICO styles based of the core Core, Pro and Magazine sites.

The grid system we use is based off of Bootstrap 4.0 — read the documentation here.

Breakpoints

More than 63% of our audience comes to us via a mobile device. We prioritize those readers by using min-width.

Wait, how do min-width breakpoints work again?

The base level css you're writing defaults to the smallest boostrap dedault breakpoint min-width:576px.

Spatial Units and how to use them

Margin and padding are based on .5rem increments. This system, inspired by Buzzfeed's solid framework used a 6 step system from
0 to 6.

Spacial units Output
0 = 0rem
05 = .25rem
1 = .5rem
2 = 1rem
3 = 1.5rem
4 = 2rem
5 = 2.5rem
6 = 3rem
Directions Meaning
m margin
p padding
b bottom
l left
r right

Now let's combine them

Spacial units and directions can be combined to deploy standard spacing between elements.

Our standard spacing between charts and section headers is 3rem. A section tag will do this for you, but you can also bring this in by adding the class of pt3.
